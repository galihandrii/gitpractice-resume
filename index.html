make ur own project

  1. git config = config ur github acc to git on ur vscode
  2. create repo on gituhb
  3. git init = initialize to ur repo
  4. coding
  5. git add = move ur coding to staging area
  6. git commit -m "title" = move from stage area to local repo
  7. git push -u origin master = ur code has been pushed to the github
  8. git status = check ur status

clone from someone repo

  1. git clone in bash
  2. git clone theurl = clone from someone repo
  3. git branch -a = check master or not
  4. git fetch = for updates branch
  5. git checkout -b "branch name" = make ur own branch
  6. git checkout namabranch = move to another branch
